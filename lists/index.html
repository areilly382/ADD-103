Description List- Crochet Terms

Slip stitch
- Put hook through stitch, yarn over once, pull through
Single Crochet
- Put hook through stitch, yarn over once, pull through stitch, yarn over again and pull through
Half double crochet
- Yarn over, put hook through stitch, yarn over, pull through stitch, yarn over and pull through
Double crochet
- Yarn over, put hook through stitch, yarn over, pull through stitch, yarn over and pull through 2, yarn over and pull through
Treble crochet
- Yarn over twice, put hook through stitch, yarn over, pull through stitch, yarn over and pull through 2, yarn over pull through 2, yarn over and pull through
